১. HTML কী? (What is HTML?)

HTML হলো HyperText Markup Language। এটি কোনো প্রোগ্রামিং ল্যাঙ্গুয়েজ নয়, বরং একটি মার্কআপ ভাষা (Markup Language), যা ওয়েব পেজ তৈরির মূল ভিত্তি হিসেবে কাজ করে। HTML এর মাধ্যমে একটি ওয়েব পেজের মৌলিক কাঠামো বা কঙ্কাল তৈরি করা হয়— যেমন হেডিং, প্যারাগ্রাফ, লিংক, টেবিল, ছবি, ভিডিও, ফর্ম ইত্যাদি।

সহজ উদাহরণ: একটি বহুতল ভবনের ভেতরের রড এবং ইটের তৈরি মূল কাঠামোটি হলো HTML, বিল্ডিংয়ের নান্দনিক রঙের ডিজাইন বা ডেকোরেশন হলো CSS, আর বিল্ডিংয়ের স্বয়ংক্রিয় লিফট বা কলিং বেলের কার্যক্ষমতা হলো JavaScript।

২. HTML এর প্রধান বৈশিষ্ট্যসমূহ

  • এটি একটি সম্পূর্ণ মার্কআপ ল্যাঙ্গুয়েজ, কোনো জটিল প্রোগ্রামিং লজিক এতে নেই।
  • HTML কোড যেকোনো আধুনিক ব্রাউজার (যেমন: Chrome, Firefox, Edge) দ্বারা পড়ে স্ক্রিনে দেখানো যায়।
  • HTML এর প্রতিটি কন্টেন্ট সুনির্দিষ্ট একটি ট্যাগ (Tag) দিয়ে নির্ধারণ করা হয়।
  • এটি ওয়েবসাইটের স্ট্রাকচার তৈরি করে, যার সাথে পরবর্তীতে CSS ও JavaScript যুক্ত করে কার্যক্ষমতা বৃদ্ধি করা হয়।

৩. HTML ট্যাগ (Tag) কী এবং এর গঠন?

• পেজে বিভিন্ন উপাদান দেখানোর জন্য ট্যাগ ব্যবহার করা হয়। ট্যাগ মূলত কৌণিক ব্র্যাকেট < > দ্বারা বেষ্টিত থাকে।

সাধারণ ট্যাগের গঠন:

<tagname>আপনার কনটেন্ট বা লেখা এখানে থাকবে</tagname>

কোডিং উদাহরণ:

<!-- প্যারাগ্রাফ ট্যাগ -->
<p>এটি একটি প্যারাগ্রাফ।</p>

<!-- হেডিং ট্যাগ -->
<h1>এটি একটি বড় হেডিং।</h1>

<!-- লিংক বা অ্যাঙ্কর ট্যাগ -->
<a href="https://example.com">এটি একটি লিংক</a>

৪. HTML এর মৌলিক গঠন (Basic Structure)

প্রতিটি আদর্শ ওয়েব পেজ একটি নির্দিষ্ট গ্লোবাল স্ট্রাকচার মেনে চলে। নিচে কোডটি দেওয়া হলো:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>আমার প্রথম ওয়েবপেজ</title>
</head>
<body>
    <h1>স্বাগতম!</h1>
    <p>এটি আমার প্রথম HTML পেজ।</p>
</body>
</html>

৫. কিছু গুরুত্বপূর্ণ HTML ট্যাগ ও তাদের কাজ

ট্যাগ (Tag) কাজ ও বিবরণ
<h1> - <h6> হেডিং বা শিরোনাম লেখার জন্য। h1 সবচেয়ে বড় এবং h6 সবচেয়ে ছোট।
<p> প্যারাগ্রাফ বা সাধারণ অনুচ্ছেদ তৈরি করার জন্য।
<a> হাইপারলিংক বা এক পেজ থেকে অন্য পেজে যাওয়ার লিংক তৈরি করতে।
<img> ويب পেজে ছবি যুক্ত করার জন্য ব্যবহৃত হয়।
<ul>, <ol>, <li> তালিকা বা লিস্ট তৈরি করতে (বুলেট পয়েন্ট এবং সংখ্যাভিত্তিক)।
<table> তথ্য ছক বা টেবিল আকারে সাজানোর জন্য।
<form> ইউজার ইনপুট বা কন্টাক্ট ফর্ম সাবমিট করার মেইন কন্টেইনার।
<div> লেআউট ডিজাইনে বিভিন্ন সেকশন ভাগ করার জন্য ব্লক কন্টেইনার।

৬. নতুন প্রজেক্ট শুরু করার বাস্তব গাইডলাইন

  1. আপনার কম্পিউটারের ডেক্সটপে একটি নতুন ফোল্ডার তৈরি করুন: MyFirstWebsite
  2. তার ভেতর একটি নতুন ফাইল তৈরি করে নাম দিন: index.html
  3. ফাইলটি আপনার কোড এডিটর (যেমন VS Code) দিয়ে ওপেন করে ওপরের কোডটি পেস্ট করুন।
  4. ফাইলটি সেভ করে ব্রাউজারে ডাবল ক্লিক করে ওপেন করলেই আপনার তৈরি করা ডেকোরেটিভ পেজটি দেখতে পাবেন!
